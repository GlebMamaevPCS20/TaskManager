## __Commands__
-_SP_ - Возвращает список запущенных процессов с id  и именем
-_Help_ - Возвращает список доступных команд
-_KP\id или \имя процесса_ - По id - закрытие конкретного процесса, по имени - закрытие дерева процессов
-_SP_ - Запускает нужное приложение из списка
-_Clear_ - Отчистка экрана
-_Exit_ - Завершение выполнения программы
